# Error Codes

| Code | Meaning             | Fix                                     |
|------|---------------------|------------------------------------------|
| 400  | Bad Request         | Check body syntax                        |

| 403  | Forbidden           | User doesn't have permission             |
| 404  | Not Found           | Endpoint or task doesn't exist           |
| 500  | Internal Server Error | Report to support or check server logs |
